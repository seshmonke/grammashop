# Фичи тарифа Premium

Черновик-в-процессе (не чистовик целиком): состав подтверждён частично —
см. пометки у каждого пункта. Ссылки — [CONCEPT.md#тарифы](CONCEPT.md#тарифы),
[STACK.md](STACK.md). Цена Premium и лимит карточек (3000, рабочий
ориентир) — там же, в CONCEPT.md.

## Подтверждено (21.07.2026)

- **Кастомная ссылка на витрину** — слаг вместо числового `seller_id` в
  `startapp` (`t.me/grammashopbot/shop?startapp=moy-magazin` вместо
  `?startapp=42`), общий бот платформы не меняется — не откатывает
  дифференциацию «без своего бота» (см.
  [CONCEPT.md#конкуренты-и-дифференциация](CONCEPT.md#конкуренты-и-дифференциация)).
  Не реализовано (нужны: колонка `slug` у продавца, уникальность,
  резолвинг по слагу в `shop.service`, UI выбора слага в профиле).

## Унаследовано от прежних Тарифов 2-3 (уже в CONCEPT.md, перенесено на Premium автоматически при слиянии тарифов — не новое решение, но стоит подтвердить состав)

- **Автоматизированная оплата покупателем через ЮKassa** — продавец
  вводит свои API-ключи, деньги идут напрямую ему →
  [CONCEPT.md#платежи](CONCEPT.md#платежи).
- **Автоматическая интеграция с СДЭК** — создание заказа и трек-номера по
  ключам продавца →
  [CONCEPT.md#логистика-и-доставка](CONCEPT.md#логистика-и-доставка).
- **Кастомизация витрины** (цвет/толщина элементов, свой фон-фото) — по
  «Скоуп v1» по-прежнему не реализуется вообще, ни на каком тарифе, до
  первого пост-MVP скоупа →
  [CONCEPT.md#кастомизация-витрины](CONCEPT.md#кастомизация-витрины).
