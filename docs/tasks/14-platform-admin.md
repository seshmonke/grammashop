### Спринт 14: платформенная админка

Цель: платформенный админ (владелец платформы) видит список продавцов со
статусом подписки и может вручную заблокировать/разблокировать магазин —
закрывает п.8 продуктовой карты, последний пункт дорожной карты v1, не
зависящий от внешних блокеров (ЮKassa-модерация/юрпакет — см. «Очередь»,
блокируют только п.3). Задачи идут в порядке зависимости: бэк (TDD),
затем фронт, затем сквозная проверка. Сформировано 20.07.2026.
