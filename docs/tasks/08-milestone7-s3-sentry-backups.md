### Спринт 8: закрыть веху 7 целиком (S3, Sentry, бэкапы на бою)

Цель: веха 7 закрыта по своему определению — S3 и Sentry работают на
бою, ночной бэкап уходит в S3, восстановление проверяется автоматически.
Задачи идут в порядке зависимости (бэкапы требуют готового S3).

**Спринт закрыт 17.07.2026** — все 4 задачи сделаны, веха 7 дорожной
карты закрыта целиком.

  веба — build-time через Vite (`VITE_SENTRY_DSN_WEB`, GitHub secret

  Vite подставляет переменные в бандл на этапе сборки, не в рантайме).
